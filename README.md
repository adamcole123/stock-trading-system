# Stock Trading System
This is the implementation of my MSc second year project, a stock trading system

# Installation

## Requirements
- Node.js > v14.15.4
- Docker
- NPM

## Commands
To get the app running, run the following command

	> cd stock-trading-system
	> docker-compose up

## Development Environment
It is not needed to install packages locally, but to avoid errors with your IDE, run the following commands

	> cd stock-trading-system
	> cd backend
	> npm install
